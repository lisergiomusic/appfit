# 🚀 AppFit - Ideias e Features Futuras

Este arquivo serve como um backlog de ideias e melhorias para o projeto AppFit. Sempre que uma nova ideia surgir, ela será registrada aqui para planejamento futuro.

---

## 📅 Roadmap de Ideias

### 🛠️ Core & Infraestrutura
- [ ] **Migração Total para Supabase**: Finalizar a transição de todos os serviços de Firebase para Supabase (Auth, Storage, Realtime).
- [ ] **Instruções Personalizadas de Base**: Permitir que o Personal edite ou remova instruções de exercícios globais, salvando uma versão customizada que será aplicada automaticamente sempre que ele prescrever esse exercício para qualquer aluno no futuro.
- [ ] **Sistema de Logs Estruturado**: Implementar uma tabela de auditoria para ações críticas do Personal.

### 🏋️‍♂️ Treinos & Performance
- [ ] **Gráficos de Evolução**: Visualização gráfica do progresso de carga por exercício para o Aluno.
- [ ] **Cronômetro de Descanso Inteligente**: Notificação push ou som quando o tempo de descanso configurado pelo Personal acabar.
- [ ] **Calculadora de 1RM**: Ferramenta dentro da visualização do exercício para estimar a carga máxima.

### 👥 Gestão de Alunos
- [ ] **Chat Integrado**: Sistema de mensagens direto entre Personal e Aluno (Supabase Realtime).
- [ ] **Anamnese Digital**: Formulário de entrada para novos alunos com assinatura digital.
- [ ] **Fotos de Evolução**: Galeria comparativa de fotos "Antes e Depois" com grade de alinhamento.

### 🎨 UI/UX & Gamificação
- [ ] **Modo Offline**: Sincronização local para que o Aluno possa treinar sem internet em academias com sinal ruim.
- [ ] **Sistema de Streaks**: Recompensas visuais para alunos que mantêm a constância nos treinos.
- [ ] **Dark/Light Mode Sync**: Opção para seguir o tema do sistema ou fixar no Dark Mode padrão.
